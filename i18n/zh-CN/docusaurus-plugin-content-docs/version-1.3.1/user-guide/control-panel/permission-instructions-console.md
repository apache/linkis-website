## 1.角色类型

- 管理员（Admin）：能够查看所有菜单入口，管理员角色权限最大，包含了历史管理员权限
- 历史管理员（historyAdmin）：除去自己提交的任务之外，还可以查看其他历史任务
- 普通用户：查看自己提交的任务信息
